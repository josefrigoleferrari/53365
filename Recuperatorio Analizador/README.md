# Recuperatorio Analizador

## Instalación

```bash
npm install
```

## Generar archivos ANTLR

```bash
npm run build
```

Los archivos generados se almacenan en la carpeta:

```text
generated/
```

## Ejecutar

```bash
npm start
```

## Funcionalidades

- Análisis léxico
- Tabla de tokens
- Análisis sintáctico
- Árbol sintáctico
- Traducción a JavaScript
- Interpretación
