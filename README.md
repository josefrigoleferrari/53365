# Instalación
Es necesario antes que nada tener instalado VS Code y la extensión ANTLR4 grammar syntax support

Abre una ventana de comandos (Cmd, PowerShell, o terminal en Linux).
Clona el proyecto desde el repositorio GitHub utilizando el siguiente comando: git clone https://github.com/josefrigoleferrari/53365.git

Si el proyecto se clonó adecuadamente veras un mensaje "done" al final, para abrirlo en VS Code escribe en la misma terminal: code .

# Probar Programa
Una vez instalado el programa para corroborar que funciona en la terminal de VS Code escripe: npm start , y ahí si el programa funciona bien mostrara si la entrada del input.txt es válida o si tiene algun error, la tabla de Tokens y la traducción a JavaScript. Para ver el arbol apretar f5 y se lo generará automaticamente.
